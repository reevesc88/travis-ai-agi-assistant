# Preferred technical stack

Use the existing stack if already started.

If no usable stack exists, build with:

* Frontend: Preact or React.
* Language: TypeScript.
* Backend: Hono or similar lightweight API.
* Database: SQLite.
* Validation: Zod.
* Icons: Lucide or similar.
* Styling: Tailwind, CSS modules, or a clean component-based CSS system.
* Routing: URL-based routing with bookmarkable pages.
* Local persistence: SQLite or clearly separated mock persistence if backend setup is incomplete.

Do not require cloud services for the first working demo.
